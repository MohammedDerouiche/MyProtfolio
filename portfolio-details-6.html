Overveiw

The GE Retailers Data Analysis project is a comprehensive case study conducted on Kaggle. The aim of this project is to showcase SQL data analysis skills to solve various marketing and sales problems for a fictional global electronic retailer. The analysis focuses on understanding customer behavior, product performance, and sales trends to provide actionable insights for business improvement.

Methodology

I utilized SQL queries to analyze various aspects of the retailer's data, including customers, products, sales, and stores. By exploring these areas, I aimed to answer specific questions and provide actionable insights for the marketing team.
